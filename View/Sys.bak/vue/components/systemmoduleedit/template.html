Route/Controller/Action Type Full Pattern Type

This type of controller uses the Opoink route/controller/action for your module.
Ex: is http://domain.com/user/edit/save.
Route: user
Controller: edit
Action: save
This will create a file User/Edit/Save.php under your module Controller directory.

This type of controller type uses the full regular expression matching of the URL.
Ex: /user/edit/:id/save where the id will be a $_GET variable parameter.
If you are using the Opoink XML layout, please use the route/controller/action type.
