Reports and Suggestions

Your comment and suggestion is important for us. Please help us improve the Opoink Framework by sending us a report and your suggestion.

Send us an email for a report at report@opoink.com please include what bug or issue is encountered, the PHP version, and database server version as well.

A screenshot of the issue will be much more appreciated.

For suggestions please email it on suggest@opoink.com.
