Cache system is always ignored in developer's mode. e.i is less compiler, it is only compiled and streamed, this may take a light to heavy server load consumption, depending on how many less files do you have in your modules, but we found this very helpful during the development mode.

Name Description Status
LESS Clear all cached less files in /Var/Less directory {{systemcacheindex.cache.less.status}}
Deployed Files This are the files inside /public/deploy{deployment_time} this will also remove the directory itself {{systemcacheindex.cache.deployed_files.status}}
XML This is the compiled XML/HTML file from your XML layout, will clear all directories and files in /Var/Layout {{systemcacheindex.cache.xml.status}}
Database The cached data from database, will clear all directories and files in /Var/db {{systemcacheindex.cache.database.status}}
